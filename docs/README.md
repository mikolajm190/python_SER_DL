# python SER using DL

Bachelor's degree repo. It contains mainly jupyter notebooks used for speech emotion recognition using deep learning techniques.

## Info

- goal: compare traditional methods with deep learning methods at SER (Speech Emotion Recognition) task
- chosen representatives:
  - traditional - SVM (Support Vector Machine)
  - deep learning - CNN (Convolutional Neural Network)
- libraries: numpy, pandas, tensorflow, librosa
- data: TESS ([Toronoto Emotional Speech Set](https://borealisdata.ca/dataset.xhtml?persistentId=doi:10.5683/SP2/E8H2MF))
- results:

|method|accuracy|
|---|---|
|SVM|93%|
|CNN|96%|
